
# 9f

## 安装
```
npm install 9f -g
```
## 构建项目
```
9f init <template-name> [project-name]

```

## 常用命令

```
9f add | a	// 添加模板

9f list | l 	// 列出模板

9f remove | r <template-name> // 删除模板

9f update | u <template-name> // 更新模板

9f reset // 重置模板

9f init | i <template-name> [project-name] // 构建项目

```
